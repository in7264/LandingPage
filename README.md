## Getting Started

Follow these steps to run the project locally.

### 1. Clone the repository

```bash
git clone https://github.com/in7264/LandingPage.git
cd LandingPage
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run the project locally

```bash
npm start
```

After open in browser:

```
http://localhost:3000
```
